# Data

This directory holds the experimental data produced by `notebooks/01_data_collection.ipynb`.

## `rich_data_a100.json`

The complete record of all 425 runs (400 speculative + 25 baseline). Each entry contains:

| Field             | Type    | Description                                                          |
| ----------------- | ------- | -------------------------------------------------------------------- |
| `prompt_id`       | int     | Index into the 25-prompt corpus (0-24).                              |
| `category`        | string  | One of: `qa`, `code`, `summary`, `reasoning`, `dialog`.              |
| `k`               | int     | Speculation length used for this run (1, 2, 3, 4, 5, 6, 8, 10, or 0 for baseline). |
| `trial`           | int     | Trial index (1 or 2).                                                |
| `throughput`      | float   | Tokens per second.                                                   |
| `total_latency`   | float   | Wall-clock time in seconds.                                          |
| `tokens_generated`| int     | Number of new tokens emitted.                                        |
| `accept_pattern`  | list    | Per-position accept/reject for each speculation round.               |
| `peak_memory_gb`  | float   | Peak GPU memory in GiB.                                              |
| `avg_power_w`     | float   | Average GPU power draw in watts (sampled from NVML).                 |

## Reproducing

The dataset is regenerated by running `notebooks/01_data_collection.ipynb` end-to-end on an A100 instance. Approximate runtime: 90 minutes.

If you only want the analysis, the pre-computed JSON is also distributed alongside the notebooks via the project's release page.
